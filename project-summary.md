# Project Summary

My project is a Workout Tracker App. The app allows users to create workouts and add exercises to each workout.

I created a relational PostgreSQL database with three tables: users, workouts, and exercises. The users table stores user information. The workouts table stores workout sessions connected to a user. The exercises table stores exercises connected to a workout.

I hosted the database on Supabase and used Postman to test data retrieval and data creation. This project shows how a frontend app can connect to a backend database and perform CRUD operations.
